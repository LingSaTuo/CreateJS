App

com.lingsatuo.openapi.App 用于文件的打开，应用的跳转，应用的卸载等操作

App(context)

实例化 var app = new com.lingsatuo.openapi.App(libs_inthis)

viewFile(pathString)

用其他应用打开位于path的文件

var app = new com.lingsatuo.openapi.App(libs_inthis)
app.viewFile("/sdcrad/a.txt")
uninstall(packageName)

卸载应用

var app = new com.lingsatuo.openapi.App(libs_inthis)
app.uninstall("com.lingsatuo.createjs")
editFile(pathString)

调用其它应用编辑文件

var app = new com.lingsatuo.openapi.App(libs_inthis)
app.editFile("/sdcrad/a.txt")
openUrl(uriString)

调用浏览器打开指定网址

var app = new com.lingsatuo.openapi.App(libs_inthis)
app.openUrl("http://www.baidu.com")// "http://"可以省略
getAppName(packageName)

通过包名获取APP的名字

var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.getAppName("com.lingsatuo.createjs")//CreateJS
getPackageName(nameString)

通过名字获取包名（如果多个应用名字一样，则返回第一个获取到的）

var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.getPackageName("CreateJS")//com.lingsatuo.createjs
launchPackage(packageName)

通过包名启动相应的app

var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.launchPackage("com.lingsatuo.createjs")
launchApp(name)

通过app名字启动相应APP，（APP重名则启动最先获取到的）

var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.launchApp("CreateJS")
isHome()

判断是否在桌面 ，返回值为boolean值

var app = new com.lingsatuo.openapi.App(libs_inthis)
var ishome = app.isHome()

getCurrentPkgName()

获取当前app的包名，该方法需要开启无障碍服务，所以需要调用libs_inthis.startAS()

libs_inthis.startAS()
var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.getCurrentPkgName()

getCurrentPkgName(context)

该方法不需要无障碍，但获取不准确，（通过进程名称获取，所以不准确）

var app = new com.lingsatuo.openapi.App(libs_inthis)
var name = app.getCurrentPkgName(libs_inthis)
