非映射类 NodeInfo

该类需要用 CreateJSService 获取

getNodeInfo()

返回值是android.view.accessibility.AccessibilityNodeInfo具体API参见安卓文档

var info = NodeInfo.getNodeInfo()
findViewById()

通过ID获取控件集合，返回值是List<NodeInfo> 

var list = NodeInfo.findViewById("com.lingsatuo.satuo:id/id名字")
getPackageName()

相当于android.view.accessibility.AccessibilityNodeInfo.getPackageName()，返回值是CharSequence类型

var name = NodeInfo.getPackageName()
getParent()

相当于android.view.accessibility.AccessibilityNodeInfo.getParent()，返回值是NodeInfo类型，当异常时返回null

 var info = NodeInfo.getParent()
getChild(intDex)

相当于android.view.accessibility.AccessibilityNodeInfo.getChild(int)，返回值是NodeInfo类型，当异常时返回null

 var info = NodeInfo.getChild(0)
getChildCount()

相当于android.view.accessibility.AccessibilityNodeInfo.getChildCount()，返回子控件数量

 var count = NodeInfo.getChildCount()
getText()

相当于android.view.accessibility.AccessibilityNodeInfo.getText()，返回值是控件上的文字内容

 var count = NodeInfo.getText()
getViewName()

相当于android.view.accessibility.AccessibilityNodeInfo.getViewIdResourceName()，返回值是String类型，虽然大多数控件都会返回null

 var name = NodeInfo.getViewName()
getWindow()

相当于android.view.accessibility.AccessibilityNodeInfo.getWindow()，返回值为com.lingsatuo.other.accessibilityUtils.WindowInfo，具体API参见WindowInfo

var windowinfo = NodeInfo.getWindow()
isChecked()

判断View是否选中，返回值为boolean

var checked = NodeIno.isChecked()
isClickable()

判断View是否可点击，返回值为boolean

var click = NodeIno.isClickable()
toString()

相当于android.view.accessibility.AccessibilityNodeInfo.toString()

var string = NodeInfo.toString()
recycle()

相当于android.view.accessibility.AccessibilityNodeInfo.recycle()

refresh()

相当于android.view.accessibility.AccessibilityNodeInfo.refresh()

click()

模拟点击，返回值为是否点击成功 boolean

NodeInfo.click()
copy()

复制，返回值为是否复制成功

NodeInfo.copy()
cut()

剪切，返回值为是否剪切成功

NodeInfo.cut()
longClick()

模拟长按，返回值为是否长按成功

NodeIno.longClick()
paste()

粘贴，返回是否成功

scrollUp()

模拟页面上滑，返回值是否成功

scrollDown()

模拟页面下滑，返回值是否成功

findFocus(int)

相当于 android.view.accessibility.AccessibilityNodeInfo.findFocus(int)，但返回值是NodeInfo

focusSearch(int)

相当于 android.view.accessibility.AccessibilityNodeInfo.focusSearch(int)，但返回值是NodeInfo
