非映射API (Object)

该类位于com.lingsatuo.openapi.Object

在JAVA里 我们可以定义 int[] byte[] char[]等，CJS中可以使用以下方法定义这些

getByteArray(sizeInt)

    var obj = new com.lingsatuo.openapi.Object()
    var a = obj.getByteArray(1024)//相当于JAVA byte[] a = new byte[1024]

getIntArray(sizeInt)

    var obj = new com.lingsatuo.openapi.Object()
    var a = obj.getIntArray(1024)//相当于JAVA int[] a = new int[1024]

getCharArray(sizeInt)

    var obj = new com.lingsatuo.openapi.Object()
    var a = obj.getCharArray(1024)//相当于JAVA char[] a = new char[1024]

getBooleanArray(sizeInt)

    var obj = new com.lingsatuo.openapi.Object()
